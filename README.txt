Penjelasan singkat:
Project ini pada dasarnya adalah upaya untuk mengubah cara kita mengelola server dari metode manual yang menjadi sistem otomatis, atau yang biasa disebut dengan Infrastructure as Code. Dalam kegiatan ini, kita berperan sebagai arsitek yang menyusun instruksi dalam sebuah file bernama Ansible Playbook. Alih-alih kita masuk ke setiap server satu per satu untuk mengetikkan perintah instalasi, kita cukup memberikan satu komando dari komputer pusat. Fungsi utama dari sistem ini adalah untuk menciptakan infrastruktur yang bisa diduplikasi secara instan dan bebas dari kesalahan manusia.
